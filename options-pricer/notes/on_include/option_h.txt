Say you have two files that both include option.h:

// black_scholes.h
#include "option.h"

// monte_carlo.h
#include "option.h"

Then in main.cpp:
#include "black_scholes.h"
#include "monte_carlo.h"

Now Option is defined twice in main.cpp — the compiler sees two copies and throws an error: "redefinition of struct Option."

#pragma once prevents that. It says "no matter how many times this file gets included, only process it once."
